Every time you open the cards, there will be a counter showing you how many guess you have tried. Adding counter to your game, you have to add another textview and create function for counting how many time have opened the pair of card.

Description below show the steps for producing the counter:

Add New TextView To The Screen

First step for creating the counter, you need to create a textview for displaying the text. Since your application name is MemoryGame, you need to do the following step:
  • Go to MemoryGame -> res -> layout -> main.xml
  • Once you have open the file main.xml press Ctrl + F to find keyword TableRow01
  • Type the code hightligh in red square to the top of TableRow1 (as shown in images)
  • Click on Save when you finish

Make The Counter Work

The next step is to add counter to the code:

  • Go to MemoryGame -> src -> com.aviy.memory -> Manager.java
  • Once you have open the file Manager.java press Ctrl + F to find keyword newGame
  • Add the code highlighted in red box to the bottom of the function
  • Inside function setText, copy the Khmer unicode that you have converted in chapter Converting Khmer Unicode to it.
  • Now you have show counter when initialize the game, block of code above mean to set counter to 0
  • On your keyboard press Ctrl + F again and search for keyword turnCard
  • Copy the red highlighted code to the location showing in the image above.
  • Click Save when you finish

Now deploy your application again, you will see the counter in your game.
