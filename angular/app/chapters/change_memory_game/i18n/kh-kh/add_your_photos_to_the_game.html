ដាក់រូបថតរបស់អ្នកក្នុងហ្កេម

ចាប់ផ្ដើមសាកធ្វើអ្វីមួុយដែលប្លែក និង គួរអោយចង់លេង។ យើងនឹងដាក់បញ្ចូលរូបថតរបស់យើងទៅក្នុងប្រពន្ធ័ចងចាំរបស់ហ្កេម ដើម្បីដាក់បង្ហាញនៅក្នុងកាតក្រលាការូ។

សូមអនុវត្តន៍តាមការបង្ហាញជាលំដាប់នេះ ដើម្បីកែររចនាកូតហ្កេមរបស់អ្នក៖

  • ចំលងរូបថតដែលអ្នកបានកែរសំរួលនៅក្នុងជំពូក​(ថតរូប និង ការកែររូប) ទៅក្នុងប្រអប់ YourProject -> MemoryGame -> res -> drawable
  • បើក Eclipse និង ចូលទៅក្នុង ទីតាំងការងារគំរោង
  • ដោយសារ គំរោងរបស់អ្នក ឈ្មោះ MemoryGame ដូច្នេះអ្នកអាចចួលក្នុងប្រអប់នេះ
  • MemoryGame -> src -> com.aviy.memory -> Manager.java
  • នៅលើ keyboard, ចុច Ctrl + F , ហើយសរសេ private void loadImages ចុច Find
  • ពេលដែលរកឃើញ, សូមសរសេ images.add(getResources().getDrawable(R.drawable.yourPhotoName));
  • ពេលដែលអ្នកសរសេចប់, ចុច Ctrl + S ដើម្បីរក្សាទុក
  • អ្នកអាចបង្កើតឯកសារ .apk file ដោយធ្វើតាមជំពូក (Deploy Android App -> Create APK File) ឬ ដំនើរការ emulator ដោយធ្វើតាមជំពូក (Making An Android Project -> ជំហ៊ានទី 4 )
