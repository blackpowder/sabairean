Making An Android Project

The first step in creating a simple Android Application is to create and Android project. Eclipse organizes your work into “projects” and by designating your project as an Android project. The following screen shot should be display on Eclipse Juno. You will have different screen shot base on your version of Eclipse.

Create A Project

To create the project, you have to follow the following step:

  • Click on File -> New -> Android Application Project
  • The following window will appear
  • Type the project name "MyProject" -> Click Next
  • Workspace indicate the location where you store your project
  • Choose the launcher icon and click next
  • Choose the Activity, here we choose blank activity
  • Put your activity name, here we choose MainActivity
  • Click finish. Now you have created an android project that display word "Hello World" on the welcome screen.
