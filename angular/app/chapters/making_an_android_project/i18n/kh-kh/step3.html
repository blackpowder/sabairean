Making An Android Virtual Device (AVD)

The Android SDK provides an emulator, which emulates a device with an ARM CPU running an Android operating system (OS), for running Android programs on your PC.

You can create an Android Virtual Device by clicking on Icon display in the window below:

  • The AVD list will appear on the window below, Click on New to create new AVD
  • In this window, fill your AVD Name and choose the Device Type (Here I choose Nexus S, the rest information will be filled for you)
  • Click on OK to finish.
