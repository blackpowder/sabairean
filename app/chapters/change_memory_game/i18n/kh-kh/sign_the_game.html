--Sign The Game

Let's style sign the game for you own. You can create multiple .apk file and put your name on the welcome screen or display your own photo.

Please follow this step by step tutorial to start customize your code:

Show Your Name On The Welcome Screen

  • Open Eclipse and go to your project workspace
  • Since your project name is MemoryGame, so go to the following folder
  • MemoryGame -> res -> values -> string.xml
  • Inside the red circle, copy the Khmer unicode you have converted in chapter Converting Khmer Unicode
  • Once you have finish, press Ctrl + S to save your project
  • You can create the .apk file by following the chapter (Deploy Android App -> Create APK File) or run on the emulator by following chapter (Making An Android Project -> Step 4)

Add Your Photo To The Home screen

  • Open Eclipse and go to your project workspace
  • Since your project name is MemoryGame, so go to the following folder
  • MemoryGame -> res -> layout -> main.xml
  • On your keyboard, press Ctrl + F and search for TableRow03
  • Inside your the red circle, replace card1 with your photo name
  • Once you have finish, press Ctrl + S to save your project
  • You can create the .apk file by following the chapter (Deploy Android App -> Create APK File) or run on the emulator by following chapter (Making An Android Project -> Step 4)
