Running a Program on an Android Device

You can also run the program you just created on most android device. These following steps show you how to deploy your application.

  • Connect your device to your PC with a USB cable.
  • Turn on the USB debugging in you Android device.
  • Right click on your project.
  • Select run as Android Application.

If an AVD is configured or is running, the Android Device Chooser will appear, displaying both the Android device you have connected and the AVD.

If this window doesn't appear, you need to do the following step:

  • Right click on your project
  • Select Run As
  • Choose Run Configurations...

Enable USB Debugging on an Android Device

USB Debugging need to be manually enable by following this step:

  • Click on Application Setting
  • Select Application
  • Select Development
  • Turn on the USB Debugging
