Create the APK File

If you want to give someone your application to try out, you can just give them the copy of the .apk file in the bin directory of the project file hierarchy. To do this, you need to sign the application by using a self-signed certificate.

Notice that, it is not the same as deploying, but it is the procedure of exporting.

In this example, we use the TestActivity project, but you can use any application. The step for exporting is shown as followed:

  • Right click on your project and select export
  • Choose Android
  • Choose Export Android Application
  • If your application don't have any error, the window below will appear
  • Next you will select the keystore, the keystore is the file holding your key
  • Choose the location where you want to store your key and fill the password
  • Now it is step to create the self-signed certificate
  • Filled the required informations
  • Click Finish

Now you have the .apk file, you can copy this file to the android device and click to install. There are several tools that you can browse to find where your .apk file locate such as file manager.
