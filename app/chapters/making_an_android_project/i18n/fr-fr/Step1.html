Créer un projet pour Android

La première étape dans la création d'une simple application pour Android est de créer "un projet pour Android". Eclipse organise ton travail en "projet" et en désignant ton projet comme un projet pour Android. La capture d'écran suivante devrait apparaitre sur Eclipse Juno. Tu vas avoir différentes captures d'écrans basé sur notre version d'Eclipse.

Créer un projet

Pour créer un projet, tu dois suivre les étapes suivantes :

  • Clique sur "Fichier" (File) -> "Nouveau" (New) -> "Projet d'application pour Android" (Android Application Project)
  • La fenêtre suivante va apparaitre
  • Tape le nom du projet "MyProject" -> Clique sur "Suivant" (Next)
  • Workspace indique l'emplacement où se trouve ton projet
  • Choisis "l'icone de lancement" (launcher icon) ..... et appuis sur "suivant" (next)
  • Choisis une activité, ici nous choisissons "blank activity" <
  • Indique un nom à ton activité (activity name), ici nous choisissons "Mainactivity" <
  • Clique sur "finir" (finish). maintenant tu as créé un projet pour Android qui affiche "Hello World" sur l'écran d'accueil. <
