Add Your Photos The The Game

Let's start doing the interesting thing. We will add our own photos to the memory game to display them in the grid card.

Please follow this step by step tutorial to start customize your code:

  • Copy the photos you have converted in chapter (taking and resizing photo) to folder YourProject -> MemoryGame -> res -> drawable
  • Open Eclipse and go the your project workspace
  • Since our project's name is MemoryGame so you can go to this folder
  • MemoryGame -> src -> com.aviy.memory -> Manager.java
  • On you keyboard, press Ctrl + F , type this word private void loadImages and click Find
  • Once you have found, type this line of code images.add(getResources().getDrawable(R.drawable.yourPhotoName));
  • Once you have finish, press Ctrl + S to save your project
  • You can create the .apk file by following the chapter (Deploy Android App -> Create APK File) or run on the emulator by following chapter (Making An Android Project -> Step 4)
