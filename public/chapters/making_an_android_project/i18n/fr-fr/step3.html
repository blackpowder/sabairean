Créer un Android Virtual Device (AVD)

***

The Android SDK fournit un émulateur qui immite un dispositif avec un ARM CPU

Tu peux créer un Android Virtual Device en cliquand sur l'icone "afficher" (display) dans la fenêtre ci-dessous. ***************** Android Virtual Device ??

  • La liste AVD va s'afficher sur la fenêtre ******, Clique sur "Nouveau" (new) pour créer un nouveau AVD
  • Dans cette fenêtre, écris le nom de ton AVD et choisis le type de Device (Ici j'ai choisis Nexus S, le reste des informations vont être écrit automatiquement pour toi)
  • Clique sur "ok" pour finir.
